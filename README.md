# Lil Bucks — Strategic Intelligence Engine (demo spec)

A single-file React app demonstrating a weekly intelligence + content-brief engine, built around [Lil Bucks](https://lovelilbucks.com) — the sprouted-buckwheat snack brand. One populated week (May 18–24, 2026) of intelligence covering Cultural Pulse, category + policy trends, competitor watch, brand ambassadors, and 12 weekly content briefs with full caption variants, shot lists, and source citations.

Built by [OBSOLETE](https://obsolete.systems) as a working spec of the kind of marketing-operating-system OBSOLETE installs for consumer brands. All brand information referenced is publicly available; no proprietary information from Lil Bucks was used.

## What's inside

- **Welcome Issue** — magazine-style cover with the week's strategic thesis, 3 verb-led moves, and 5 evidence stats sourced from this week's category news.
- **Intelligence Brief** with four sub-tabs:
  - *Cultural Pulse* — 10 hooks (memes, viral recipes, entertainment moments, cultural conversations) with platform-specific willing-to-ship plays.
  - *Category + Policy* — 10 trends across snacking, regen-ag, fibermaxxing, hot-honey wave, ROC certification.
  - *Competitor Watch* — 7 competitor cards with editorial 3-column "what happened / what's coming / the plays" lede.
  - *Brand Ambassadors* — 7 creator + collaborator profiles.
- **Content Briefs** — 12 briefs grouped by content pillar (FUEL · THE BOWL · SOIL · CRUNCH BUNCH), each with 3 alt hooks, 3 caption variants (direct/warm/punchy), visual + audio direction, music references, scene-by-scene shot lists for video briefs.
- **Send to Studio** action on every intelligence card and brief — the intel-to-ideation handoff that makes the engine feel continuous.
- **How It Works modal** — the four-loop product framing (Train → Orchestrate → Run → Learn), the six engine agents, and the decisions the engine made this week.

## Tech

Single-file React (no build step). React 18 + Babel standalone + inline CSS via `<style>`. Click `index.html` in a browser; done.

## Status
